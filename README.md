# GrowIQ Digital

Futuristic agency hub (Next.js App Router). Local-first lead capture (no outbound sends).

## Run

```bash
npm install
npm run dev
```

## Lead capture

- UI: `/contact`
- API: `/api/leads`
- Storage: `data/leads.json` (created automatically)

No outbound sending is enabled. This is intentionally approval-gated.

## Deploy (Replit)

- Import from GitHub (source of truth)
- Run command: `npm run dev` (for preview) or Replit Deploy as configured
- Add custom domain later (approval-gated)
